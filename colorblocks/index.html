Colors and Blocks

  • Here is an example of adding and assining numbers to color blocks. Hover of the block to see assigned number.

  • Click on each block to change the color.

  • Double clicking on an even block will remove the block to the right of the even square. If there is not block to the right, you will be prompt as sush.

  • Double clicking on an odd block will remove the block to the left of the odd square. If there is not block to the left,you will be prompt as sush.
